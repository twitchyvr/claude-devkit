# devlog-hook

AI-powered development journal that writes itself.

## Usage

```bash
# Install in a single repo
cd ~/code/myproject
devlog-hook --install

# Install globally for ALL repos
devlog-hook --install-global

# Run manually (summarize latest commit)
devlog-hook
```

## How It Works

1. After every `git commit`, the post-commit hook runs `devlog-hook &` in the background
2. It reads the commit hash, message, body, and file diff stats
3. Sends a prompt to Claude asking for a one-line WHY-focused summary
4. Appends the entry to `DEVLOG.md` in the repo root

Because it runs with `&`, your commit returns instantly — the DEVLOG entry appears a few seconds later.

## Output

```markdown
# Development Log

> Auto-generated by devlog-hook — AI-powered commit summaries.

- `2026-03-22 14:30` [`a1b2c3d`] Added rate limiting to prevent API abuse during peak traffic
- `2026-03-22 12:15` [`e4f5g6h`] Fixed auth token refresh to prevent socket disconnects
- `2026-03-21 19:00` [`i7j8k9l`] Enabled dark mode toggle based on user accessibility feedback
```

## Installation Details

### Per-repo install (`--install`)

Creates `.git/hooks/post-commit` (or appends to an existing one):

```bash
#!/usr/bin/env zsh
# AI devlog — auto-summarize commits
devlog-hook &
```

### Global install (`--install-global`)

1. Creates `~/.config/git/hooks/post-commit`
2. Sets `git config --global core.hooksPath ~/.config/git/hooks`

**Note:** Global `core.hooksPath` overrides per-repo hooks. If a repo needs its own hooks, use the per-repo install instead.

## Configuration

| Variable | Default | Description |
|----------|---------|-------------|
| `CLAUDE_DEVKIT_MODEL` | `haiku` | Claude model to use (haiku is fast + cheap) |

## Tips

- Add `DEVLOG.md` to your repo and commit it — it becomes a readable project history
- The AI focuses on WHY, not WHAT — "Added rate limiting to prevent API abuse" not "Added middleware to routes.ts"
- If Claude Code is unavailable, the hook falls back to using the commit message as-is
- Works great alongside conventional commits — the AI summary complements the structured message
